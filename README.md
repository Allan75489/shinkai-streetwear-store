# SHINKAI STORE 👕🔥

Landing page premium inspirada no universo anime, criada para representar uma marca moderna de streetwear geek com foco em identidade visual forte, experiência imersiva e design impactante.

---

## 🚀 Sobre o Projeto

O **SHINKAI STORE** é uma página de apresentação desenvolvida com foco em:

* Visual moderno e premium
* Estética inspirada em anime e cultura japonesa
* Layout responsivo
* Experiência visual envolvente
* Estrutura ideal para loja virtual futura

---

## 🎯 Funcionalidades

✅ Hero section impactante
✅ Catálogo de produtos em destaque
✅ Coleções temáticas
✅ Seção institucional da marca
✅ Depoimentos de clientes
✅ Newsletter para captação
✅ Rodapé profissional
✅ Navegação fluida
